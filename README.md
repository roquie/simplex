Решение симплекс- и двойственного симпекс-метода
=======

![](http://i.imgur.com/VD46yQC.png)

## Запуск

* `git clone`
* Затем запустите `./manage.sh` и всё ;)
* `docker-compose down` - остановит и убьет лишнее.
